Benvinguts a la Galeria Botànica Digital!

Demostració en viu del sistema desenvolupat com a Treball de Fi de Grau

Aquesta aplicació web és una rèplica fidel del plugin WordPress creat per integrar fàcilment la galeria botànica i el mapa interactiu a les pàgines oficials de la UAB. Utilitza les mateixes funcionalitats, dades i interfície que el sistema original, implementat amb tecnologies web obertes per garantir la màxima compatibilitat i rendiment.

Sistema de Filtres Avançat

Filtra plantes per família, tipus, colors, hàbitat, època de floració i molt més. Combina múltiples criteris per trobar exactament el que busques.

📱

Disseny Responsive

Perfectament adaptat a mòbils, tauletes i ordinadors. Experiència d'usuari consistent en qualsevol dispositiu.

🗺️

Mapa Interactiu

Localitza espècies al campus amb coordenades GPS precises, marcadors personalitzats i zones d'hàbitat geolocalitzades.

📸

Galeria Visual Rica

Imatges d'alta qualitat de flors, fulles, fruits i hàbits de creixement amb visualització lightbox avançada.

⚡

Càrrega Ràpida

Optimitzat per a màxim rendiment amb lazy-loading d'imatges i clustering intel·ligent de marcadors.

🌐

Tecnologies Obertes

Basat en WordPress, Leaflet, JSON i tecnologies web estàndard per garantir sostenibilitat a llarg termini.
